# SerialScanner

A Qt-based serial device scanning tool developed with CMake.

### Directory Structure

```shell
SerialScanner/
├── HC				#Qt		
│   ├── forms
│   ├── include
│   └── src
└── SC				#STM32F103C8
    ├── Boot
    ├── Hardware
    ├── Lib
    ├── System
    └── User

```

**Warning**: This project currently only supports Windows systems.
